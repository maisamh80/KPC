# 📦 راهنمای نصب:

## ۱- دانلود پروژه

پروژه را از GitHub دانلود کنید.
بعد فایل دریافتی را از حالت زیپ خارج کنید و فولدر نهایی را در جایی نگه دارید.


## ۲- باز کردن صفحه افزونه‌های Chrome

مرورگر Chrome را باز کنید و وارد آدرس زیر شوید:

```
chrome://extensions
```

سپس گزینه

```
Developer Mode
```

را فعال کنید.

