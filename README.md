# Kitsu Persian Calendar v1.1.0

## رابط کاربری جدید

با کلیک روی آیکون اکستنشن، Popup تنظیمات باز می‌شود.

قابلیت‌ها:

- کلید فعال/غیرفعال‌کردن اکستنشن
- فیلد آدرس Kitsu
- دکمه استفاده از آدرس تب فعلی
- ذخیره تنظیمات در `chrome.storage.sync`
- بازخوانی خودکار تب پس از ذخیره
- اجرای اکستنشن فقط روی Origin تعیین‌شده

نمونه آدرس‌های معتبر:

- `http://localhost`
- `http://192.168.1.20`
- `https://kitsu.example.com`

مسیر صفحه مهم نیست. مثلاً با تنظیم `http://localhost`، اکستنشن روی تمام مسیرهای زیر همان دامنه اجرا می‌شود.

## آیکون‌ها

پوشه `icons` در پروژه قرار دارد و manifest به فایل‌های زیر متصل شده است:

- `icons/icon16.png`
- `icons/icon32.png`
- `icons/icon48.png`
- `icons/icon128.png`

فایل‌های فعلی placeholder هستند. کافی است آیکون‌های خودتان را با همین نام و اندازه جایگزین کنید.

# 📦 راهنمای نصب:

## ۱- دانلود پروژه

پروژه را از GitHub دانلود کنید.
بعد فایل دریافتی را از حالت زیپ خارج کنید و فولدر نهایی را در جایی نگه دارید.


## ۲- باز کردن صفحه افزونه‌های Chrome

مرورگر Chrome را باز کنید و وارد آدرس زیر شوید:

```
chrome://extensions
```

سپس گزینه

```
Developer Mode
```

را فعال کنید.
