# Kitsu Persian Calendar v1.1.0


این ریپوزیتوری یک اکستنشن برای مرورگر کروم است و با استفاده از این ابزار کلیه تاریخ های میلادی در نرم افزار کنترل پروژه انیمیشن "کیتسو" به صورت تاریخ شمسی(جلالی) نمایش داده می شود. این اکستنشن صورت سورس باز، آزاد و رایگان برای عموم منتشر شده.

## ✨ امکانات:

- ✅ نمایش تقویم شمسی
- ✅ انتخاب تاریخ به صورت شمسی
- ✅ نمایش تاریخ شمسی هنگام قرار گرفتن موس روی تاریخ‌ها
- ✅ تبدیل تاریخ‌های تسک‌ها
- ✅ تبدیل بخش Schedule
- ✅ تبدیل بخش Calendar
- ✅ تبدیل فیلترهای ماه و سال
- ✅ تبدیل فرم‌های Start Date
- ✅ فعال یا غیرفعال کردن افزونه
- ✅ امکان تعیین آدرس Kitsu

## 📦 راهنمای نصب:

### ۱- دانلود پروژه

پروژه را از GitHub دانلود کنید.
بعد فایل دریافتی را از حالت زیپ خارج کنید و فولدر نهایی را در جایی نگه دارید.


### ۲- نصب اکتنشن بر روی Google Chrome

مرورگر Chrome را باز کنید و وارد آدرس زیر شوید:

```
chrome://extensions
```

سپس گزینه

```
Developer Mode
```

را فعال کنید.

روی دکمه

```
Load unpacked
```

کلیک کنید.

پوشه پروژه را انتخاب کنید.
افزونه نصب خواهد شد.

---
# ⚙️ راه‌اندازی

پس از نصب، روی آیکون افزونه کلیک کنید.
پنجره تنظیمات باز می‌شود.
کلید "فعال سازی اکستنشن" را روشن کنید.
### تعیین آدرس Kitsu:
آدرس سرور Kitsu خود را وارد کنید.
مثال:

```
http://localhost یا http://192.168.1.110:80
```
یا با استفاده از دکمه "استفاده از آدرس تب فعلی" آدرس آن را تنظیم کنید.
<br>
<br>
از این پس افزونه فقط روی همین آدرس فعال خواهد بود.
